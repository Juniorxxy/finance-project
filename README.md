# Finance App for Couples

## Visão Geral

Aplicativo mobile que permite que casais compartilhem e controlem suas finanças em um projeto compartilhado. Os usuários podem cadastrar despesas, vinculá-las a categorias, acompanhar status, tipo de despesa e método de pagamento.

### Funcionalidades do MVP:

- Cadastro de usuários e projetos compartilhados.
- Cadastro de categorias (com categorias padrão já carregadas).
- Cadastro de despesas vinculadas a categoria, usuário que paga, tipo, status e método de pagamento.
- Visualização de despesas por categoria, status e usuário.
- Dashboard inicial (planejado) para acompanhamento financeiro do casal.

---

## Stack Tecnológica

- Frontend: React Native / Expo
- Backend: Node.js + TypeScript + Express
- Banco de Dados: Supabase (PostgreSQL)
- ORM: TypeORM
- Variáveis de ambiente via `.env`
